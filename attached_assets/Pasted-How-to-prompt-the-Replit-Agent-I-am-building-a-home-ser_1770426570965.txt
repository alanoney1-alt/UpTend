Since background checks and ID verifications aren't instant, you need Webhooks. This is how the external API "calls" your Replit app to say, "Hey, Mike just passed his check!"

Code Snippet for your Replit Webhook Handler:

JavaScript
app.post('/api/webhooks/checkr', async (req, res) => {
  const event = req.body;
  
  // If the background check is completed and "clear"
  if (event.type === 'report.completed' && event.data.object.status === 'clear') {
    const workerId = event.data.object.external_id;
    
    // Update worker status in your Replit DB
    await db.query('UPDATE workers SET background_check_status = $1 WHERE id = $2', ['clear', workerId]);
    
    console.log(`Worker ${workerId} is now safety-cleared!`);
  }
  res.status(200).send('Webhook Received');
});
