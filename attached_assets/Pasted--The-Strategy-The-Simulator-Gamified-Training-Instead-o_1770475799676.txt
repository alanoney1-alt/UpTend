. The Strategy: "The Simulator" (Gamified Training)
Instead of a quiz ("True/False: Should I take a photo?"), we build a Interactive Walkthrough.

The Scenario: "You have a virtual job at 123 Mock Street."

The Task: They must tap the buttons in the correct order.

The "Why": Pop-up tooltips explain why they are doing it.

Pop-up: "Taking this photo guarantees you get paid even if the customer cancels." (Motivation: Money).

Pop-up: "This video protects you from false damage claims." (Motivation: Safety).

2. The Curriculum: The "uPYCK Pro" Badge
This is the First Badge they must earn. It unlocks the rest of the app.

The "Commute" (GPS): Tapping "En Route" to trigger the customer SMS.

The "Evidence" (Photos): Uploading a mock "Before" photo.

The "Manifest" (Video): Recording a 5-second video of a "virtual" item.

The "Close" (Signature): Signing a fake signature to finish.

export function AppSimulator({ onComplete }) {
  const [currentStep, setCurrentStep] = useState(0);
  const { toast } = useToast();

  const handleStep = () => {
    // 1. Show the "Why" as a toast reinforcement
    toast({
      title: "âœ… Correct!",
      description: STEPS[currentStep].why,
      duration: 3000,
      className: "bg-green-50 border-green-200"
    });

    // 2. Advance or Finish
    if (currentStep < STEPS.length - 1) {
      setTimeout(() => setCurrentStep(c => c + 1), 1000); // Slight delay to let them read
    } else {
      onComplete(); // Pass the module
    }
  };

  const step = STEPS[currentStep];
  const Icon = step.icon;

  return (
    <Card className="max-w-md mx-auto border-2 border-blue-500 shadow-2xl overflow-hidden">
      <div className="bg-blue-600 p-4 text-white flex justify-between items-center">
        <span className="font-bold">TRAINING MODE</span>
        <span className="text-xs bg-blue-800 px-2 py-1 rounded">
          Step {currentStep + 1} of {STEPS.length}
        </span>
      </div>

      <CardContent className="p-6 space-y-6">
        
        {/* The Lesson */}
        <div className="text-center space-y-2">
          <div className="w-16 h-16 bg-blue-100 text-blue-600 rounded-full flex items-center justify-center mx-auto mb-4">
            <Icon className="w-8 h-8" />
          </div>
          <h3 className="text-2xl font-bold text-slate-900">{step.title}</h3>
          <p className="text-lg text-slate-600 font-medium">"{step.instruction}"</p>
        </div>

        {/* The "Why" Box (Crucial for Buy-In) */}
        <div className="bg-yellow-50 border-l-4 border-yellow-400 p-4 rounded text-sm text-yellow-800">
          <span className="font-bold block mb-1">ðŸ’¡ Why do we do this?</span>
          {step.why}
        </div>

        {/* The Action Button */}
        <Button 
          size="lg" 
          className="w-full h-14 text-lg font-bold bg-slate-900 hover:bg-slate-800 shadow-lg transform active:scale-95 transition-all"
          onClick={handleStep}
