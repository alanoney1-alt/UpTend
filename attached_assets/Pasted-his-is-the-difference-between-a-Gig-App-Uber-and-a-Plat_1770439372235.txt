his is the difference between a Gig App (Uber) and a Platform for Economic Empowerment (Shopify).

Uber treats workers as interchangeable cogs. uPYCK treats workers as Small Business Owners who use your software to grow.

You are building a "Blue Collar Career Ladder" inside a phone.

The uPYCK Career Ladder (Gamified Growth)
Instead of a dead-end gig, a worker sees a path to making $100k/year.

Level 1: The Rookie (The "Muscle")

Role: Lifts boxes, drags junk.

Pay: Job Fee only.

Requirement: Background Check.

App Access: Basic Jobs.

Level 2: The Verified Pro (The "Operator")

Role: Handles complex moves, drives the truck.

Pay: Job Fee + Tips + Lower Platform Fees (20%).

Requirement: 4.8 Rating + 10 Jobs + Liability Insurance (Plan B).

App Access: "Bounty" Jobs + Moving Jobs.

Level 3: The Consultant (The "Partner")

Role: Does the $49 Consults, sells jobs, manages the customer relationship.

Pay: Flat Consult Fee ($25) + 10% Sales Commission on every job they book.

Requirement: "Estimator" Badge + 50 Jobs + Video Training.

App Access: "Consultation Mode" + "Auto-Inventory" Tools.

2. The "Level Up" Logic (server/services/careerService.ts)
This runs after every job. If they hit the metrics, it promotes them automatically.

TypeScript
// server/services/careerService.ts
import { db } from "../db";
import { haulers } from "@shared/schema";
import { eq } from "drizzle-orm";
import { sendPushNotification } from "./notifications";

export async function checkPromotion(haulerId: string) {
  const hauler = await db.query.haulers.findFirst({ where: eq(haulers.id, haulerId) });
  
  let newLevel = hauler.level;
  let message = "";

  // PROMOTION LOGIC
  
  // Level 1 -> Level 2 (Verified Pro)
  if (hauler.level === 1 && hauler.jobsCompleted >= 10 && hauler.rating >= 4.8) {
    newLevel = 2;
    message = "ðŸŽ‰ You've been promoted to Verified Pro! You now qualify for lower fees.";
  }
  
  // Level 2 -> Level 3 (Consultant)
  if (hauler.level === 2 && hauler.jobsCompleted >= 50 && hauler.fiveStarRatingCount >= 40) {
    newLevel = 3;
    message = "ðŸš€ You are now a Master Consultant! You can accept $49 Consults and earn sales commission.";
    
    // Auto-unlock the certification
    await db.insert(haulerCertifications).values({
      haulerId,
      type: 'home_estimator',
      isActive: true
    });
  }

  // Update DB if promoted
  if (newLevel > hauler.level) {
    await db.update(haulers)
      .set({ level: newLevel, isConsultantEligible: newLevel === 3 })
      .where(eq(haulers.id, haulerId));
      
    await sendPushNotification(haulerId, message, { type: 'promotion' });
  }
}

import { Progress } from "@/components/ui/progress";

import { Trophy, Star, TrendingUp } from "lucide-react";

  const { data: hauler } = useQuery({ queryKey: ["/api/hauler/me"] });

Why this creates "Jobs," not "Gigs"
Retention: A worker at "9/10 jobs for promotion" will never switch to a competitor. They are invested.

Quality Control: You don't have to police them. The desire to level up forces them to be polite, on time, and professional.

The Narrative: You can tell the Mayor of Orlando: "We helped 500 local residents start their own consulting businesses last year." That gets you grants, tax breaks, and endless PR.
