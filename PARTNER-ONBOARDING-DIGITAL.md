# Partner Digital Onboarding Playbook
## What Cap'n Does When a New Pro Signs Up

**Full SEO strategy:** See `PARTNER-SEO-PLAYBOOK.md` for the complete zero-click GBP optimization guide.
**Core principle:** 60% of searches are zero-click. The GBP IS the storefront. Treat it that way.

### Immediate (Day 1)
1. **Google Business Profile audit** — check categories, phone, description, hours, photos. Fix everything.
2. **Bing Places** — create or claim listing, sync from GBP
3. **Yelp** — create or claim business page
4. **BBB** — check if listed, create if not
5. **Angi / HomeAdvisor** — verify listing
6. **Apple Business Connect** — create listing
7. **Nextdoor Business** — create page
8. **Update UpTend partner page** — /partners/[slug] with full content
9. **Update llms.txt** — add partner as verified provider
10. **Update structured data** — add LocalBusiness schema for partner

### Week 1
11. **Website audit** — evaluate their site, note issues
12. **Social media audit** — find/document all their accounts
13. **NAP consistency check** — name, address, phone same everywhere
14. **First review request** — prompt partner to ask 3 happy customers for Google reviews
15. **Cross-link** — get partner to link to UpTend from their site

### Ongoing
16. **Monitor LLM mentions** — test ChatGPT/Perplexity quarterly for partner visibility
17. **Review monitoring** — track new reviews, respond if needed
18. **Content creation** — write service-area pages featuring partner

## UpTend Platform Checklist (Ongoing)

### Weekly
- [ ] Check Bing Webmaster for crawl errors
- [ ] Check Google Search Console for issues
- [ ] Monitor site uptime and page speed

### Monthly
- [ ] Test ChatGPT: "AC repair Orlando" — does UpTend show?
- [ ] Test Perplexity: "HVAC companies Orlando" — does UpTend show?
- [ ] Test Claude: "home services Orlando FL" — does UpTend show?
- [ ] Test Google AI Overview for target queries
- [ ] Review and update llms.txt if services changed
- [ ] Check directory listings are active and accurate
- [ ] Write 2-4 answer-engine blog posts

### Quarterly
- [ ] Full SEO audit
- [ ] Competitor visibility check (what do LLMs say about competitors?)
- [ ] Directory presence expansion (new platforms?)
- [ ] Review count check — are we growing?
